# My To-Do list
My first to-do list application developed with HTML, CSS, and JavaScript.

## Description
This application allows you to create, complete, and delete tasks. Additionally, tasks are automatically saved in the browser using localStorage, so they are not lost when closing the page.

## What does the page?
- Create tasks
- Delete tasks
- Mark tasks as completed
- Add tasks by pressing the Enter key.
- Validate empty tasks
- Save tasks using localStore
